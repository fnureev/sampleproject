# Sample Project
Просто небольшой пример кода, приложение к резюме.
Код писался на заказ, но не пригодился.

Ищет сайты с заданнами доменами в их цепочке редиректов.

## Использование
1. php create.phar.php - создаёт parser.phar
2. php parser.phar parse - парсит список сайтов с liveinternet и сохраняет их в in.txt
3. php parser.phar check - проверяет сайты из in.txt, ищет у них среди редиректов нужные домены, сохраняет результат в out.txt
